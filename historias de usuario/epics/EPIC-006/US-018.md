# US-018: Autorización de consulta de datos tributarios

**Épica:** [EPIC-006](EPIC-006.md) - Autorizaciones de consulta de datos

## Historia de usuario
**Como** ciudadano  
**Quiero** autorizar o denegar explícitamente la consulta de mis datos tributarios  
**Para** controlar el acceso a mi información fiscal

## Descripción funcional
Para las autorizaciones de tipo "únicamente si consiente", el ciudadano debe indicar explícitamente si autoriza o no la consulta de datos. No existe un valor por defecto; debe tomar una decisión activa.

Si no autoriza la consulta de datos tributarios, deberá aportar la documentación correspondiente en el paso de documentos.

## Criterios de aceptación

### CA-1: Pregunta explícita
**Dado** que existen autorizaciones de tipo "únicamente si consiente"  
**Cuando** se muestra la sección de datos tributarios  
**Entonces** debe presentar una pregunta clara: "¿Autorizas a la Administración a consultar tus datos?" con campo obligatorio

### CA-2: Opciones de respuesta
**Dado** que se muestra la pregunta de autorización  
**Cuando** el ciudadano visualiza las opciones  
**Entonces** debe ver dos opciones excluyentes:
- "Autorizo la consulta de mis datos tributarios"
- "No autorizo la consulta de mis datos tributarios"

### CA-3: Sin selección por defecto
**Dado** que el ciudadano accede al paso de autorizaciones por primera vez  
**Cuando** visualiza la pregunta de datos tributarios  
**Entonces** ninguna opción debe estar preseleccionada

### CA-4: Selección obligatoria
**Dado** que el ciudadano no ha seleccionado ninguna opción  
**Cuando** intenta continuar al siguiente paso  
**Entonces** el sistema debe mostrar un mensaje de validación indicando que es obligatorio tomar una decisión

### CA-5: Mensaje informativo al no autorizar
**Dado** que el ciudadano selecciona "No autorizo la consulta"  
**Cuando** marca esta opción  
**Entonces** debe mostrarse un mensaje indicando: "Deberás aportar la documentación cuya consulta no autorices"

### CA-6: Registro de autorización
**Dado** que el ciudadano selecciona una opción  
**Cuando** continúa con la tramitación  
**Entonces** el sistema debe registrar la decisión (autoriza/no autoriza) para cada dato tributario

### CA-7: Documentación obligatoria si no autoriza
**Dado** que el ciudadano no autorizó la consulta de datos tributarios  
**Cuando** accede al paso de documentos  
**Entonces** los documentos relacionados con datos tributarios deben aparecer como obligatorios

## Reglas de negocio
- **RN-1**: La autorización de datos tributarios es siempre obligatoria (debe seleccionar sí o no)
- **RN-2**: No existe valor por defecto; el ciudadano debe decidir explícitamente
- **RN-3**: Si no autoriza, debe aportar la documentación correspondiente
- **RN-4**: La decisión se registra junto con la solicitud
