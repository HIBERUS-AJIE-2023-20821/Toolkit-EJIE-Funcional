# US-011: Selección y carga de otros documentos

## ID
US-011

## Título
Selección y carga de otros documentos

## Como
Ciudadano solicitante

## Quiero
Seleccionar y cargar documentos adicionales

## Para
Aportar documentación complementaria que considero necesaria para mi solicitud

## Descripción funcional

En el paso 4 de la solicitud, el ciudadano puede aportar documentación adicional bajo la categoría "Otros documentos". La funcionalidad permite seleccionar archivos desde el dispositivo del ciudadano. El ciudadano puede repetir esta operación tantas veces como necesite para aportar múltiples documentos adicionales.

Antes de finalizar la carga del documento, el sistema solicita información adicional obligatoria (tipología de documento y título) que debe completarse para que el documento quede correctamente registrado en el expediente.

## Flujo principal

1. Ciudadano accede al paso 4 de la solicitud
2. Ciudadano localiza sección "Otros documentos"
3. Ciudadano hace clic en opción para añadir otro documento
4. Sistema muestra selector de archivos
5. Ciudadano selecciona archivo desde su dispositivo
6. Sistema solicita información adicional del documento (tipología y título)
7. Ciudadano completa información requerida
8. Sistema carga el documento con la información asociada
9. Documento queda registrado en la sección "Otros documentos"
10. Ciudadano puede repetir el proceso para añadir más documentos

## Criterios de aceptación

### Escenario 1: Selección de documento

**Dado** el paso 4 de la solicitud activo  
**Cuando** el ciudadano hace clic en añadir otro documento  
**Entonces** el sistema muestra el selector de archivos  
**Y** permite seleccionar un archivo desde el dispositivo

### Escenario 2: Solicitud de información antes de carga

**Dado** un documento seleccionado desde el dispositivo  
**Cuando** el ciudadano confirma la selección  
**Entonces** el sistema solicita información adicional obligatoria  
**Y** el documento no se carga hasta completar la información requerida

### Escenario 3: Aportación múltiple de documentos

**Dado** un documento ya cargado en "Otros documentos"  
**Cuando** el ciudadano añade un nuevo documento  
**Entonces** el sistema permite repetir el proceso completo  
**Y** mantiene todos los documentos previamente cargados  
**Y** no existe límite para la cantidad de documentos a aportar

### Escenario 4: Visualización de documentos cargados

**Dado** documentos cargados en "Otros documentos"  
**Cuando** el ciudadano revisa el paso 4  
**Entonces** todos los documentos aportados aparecen listados  
**Y** cada documento muestra la información asociada introducida
