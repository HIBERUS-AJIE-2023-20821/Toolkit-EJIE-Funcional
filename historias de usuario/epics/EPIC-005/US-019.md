# US-019: Selección de modalidad de tramitación profesional

## ID
US-019

## Título
Selección de modalidad de tramitación profesional

## Como
Profesional que tramita una solicitud

## Quiero
Seleccionar si voy a tramitar en mi nombre, en nombre de otro o como autorizado de una empresa

## Para
Que el sistema registre correctamente quién es el interesado y quién tramita

## Descripción funcional

Cuando un profesional decide continuar con perfil profesional, debe indicar la modalidad de tramitación entre dos opciones:

**Para mí**: El profesional realiza la tramitación en su propio nombre, siendo él mismo el interesado de la solicitud.

**Para otro**: El profesional tramita en nombre de una persona física o jurídica diferente, actuando como representante. Esta modalidad incluye todas las casuísticas de representación: representación con documento acreditativo, o actuación como autorizado de empresa mediante registro de apoderamientos previo.

Esta selección determina cómo se solicitan los datos del interesado, qué documentación es necesaria, y cómo se gestionan las notificaciones del expediente.

## Flujo principal

1. Usuario selecciona "Continuar como profesional" en ventana informativa
2. Sistema muestra pantalla con las dos modalidades de tramitación
3. Sistema presenta opciones claramente identificadas: Para mí, Para otro
4. Profesional selecciona una modalidad
5. Si selecciona "Para mí", sistema continúa con tramitación utilizando datos propios como interesado
6. Si selecciona "Para otro", sistema solicita datos del interesado representado
7. Profesional puede navegar hacia atrás para volver a ventana informativa

## Criterios de aceptación

### Escenario 1: Presentación de modalidades

**Dado** un profesional que seleccionó continuar como profesional  
**Cuando** finaliza la ventana informativa  
**Entonces** se muestra pantalla con dos modalidades claramente identificadas  
**Y** presenta opciones: Para mí, Para otro

### Escenario 2: Selección "Para mí"

**Dado** las opciones de modalidad de tramitación  
**Cuando** el profesional selecciona "Para mí"  
**Entonces** continúa con tramitación utilizando sus propios datos como interesado

### Escenario 3: Selección "Para otro"

**Dado** las opciones de modalidad de tramitación  
**Cuando** el profesional selecciona "Para otro"  
**Entonces** el sistema solicita los datos del interesado representado

### Escenario 4: Navegación hacia atrás

**Dado** la pantalla de selección de modalidad  
**Cuando** el profesional selecciona volver  
**Entonces** regresa a la ventana de información profesional
