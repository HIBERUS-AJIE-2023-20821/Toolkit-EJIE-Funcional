# US-018: Identificación y notificación de procedimiento profesional

## ID
US-018

## Título
Identificación y notificación de procedimiento profesional

## Como
Usuario que inicia un procedimiento

## Quiero
Ser informado si el procedimiento está orientado a profesionales específicos

## Para
Decidir si deseo continuar como ciudadano o como profesional, entendiendo la naturaleza técnica del trámite

## Descripción funcional

Al iniciar una solicitud de un procedimiento marcado como profesional, el sistema presenta al usuario información sobre la naturaleza técnica del trámite antes de comenzar la cumplimentación. Esta información aclara que el procedimiento está orientado a profesionales específicos pero que cualquier ciudadano puede cumplimentarlo si lo desea.

El usuario puede elegir entre tres opciones: salir de la tramitación, continuar como ciudadano con el flujo estándar, o continuar como profesional seleccionando una modalidad de tramitación profesional. La decisión del usuario determina el resto del flujo de tramitación.

Si el procedimiento no está marcado como profesional, se continúa directamente con el flujo estándar de tramitación sin mostrar información adicional.

## Flujo principal

1. Usuario selecciona e inicia un procedimiento
2. Sistema verifica si el procedimiento está marcado como profesional
3. Si está marcado como profesional, sistema muestra ventana informativa antes del primer paso
4. Ventana presenta información sobre naturaleza técnica del procedimiento
5. Sistema ofrece tres opciones claramente identificadas: Salir, Continuar como ciudadano, Continuar como profesional
6. Usuario selecciona una opción
7. Si selecciona "Salir", abandona la tramitación y vuelve al inicio
8. Si selecciona "Continuar como ciudadano", continúa con flujo estándar de tramitación
9. Si selecciona "Continuar como profesional", sistema muestra opciones de modalidad de tramitación profesional
10. Si no está marcado como profesional, continúa con flujo estándar directamente

## Criterios de aceptación

### Escenario 1: Procedimiento profesional - Presentación de información

**Dado** un usuario que inicia un procedimiento marcado como profesional  
**Cuando** el sistema carga la tramitación  
**Entonces** muestra una ventana informativa antes del primer paso  
**Y** la ventana indica claramente que es un procedimiento para profesionales  
**Y** aclara que cualquier ciudadano puede cumplimentarlo

### Escenario 2: Opción "Salir"

**Dado** la ventana informativa de procedimiento profesional  
**Cuando** el usuario selecciona "Salir"  
**Entonces** abandona la tramitación  
**Y** vuelve a la pantalla de inicio

### Escenario 3: Opción "Continuar como ciudadano"

**Dado** la ventana informativa de procedimiento profesional  
**Cuando** el usuario selecciona "Continuar como ciudadano"  
**Entonces** continúa con el flujo de tramitación estándar  
**Y** no se aplican reglas específicas de profesionales

### Escenario 4: Opción "Continuar como profesional"

**Dado** la ventana informativa de procedimiento profesional  
**Cuando** el usuario selecciona "Continuar como profesional"  
**Entonces** el sistema muestra las opciones de modalidad de tramitación profesional

### Escenario 5: Procedimiento no profesional

**Dado** un usuario que inicia un procedimiento no marcado como profesional  
**Cuando** el sistema carga la tramitación  
**Entonces** continúa con el flujo estándar sin mostrar ventana informativa
